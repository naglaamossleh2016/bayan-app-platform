السلوك على الشاشات المختلفة

ملاحظات هامة:
  • النصوص الأساسية: تستخدم line-height: 1.85 لجميع الشاشات لتحسين القراءة
  • العناوين: تتقلص تلقائياً بواسطة Bootstrap على الشاشات الصغيرة
  • شريط التنقل (Navbar): يقل حجم العلامة التجارية إلى 1rem على شاشات أصغر من 576px
  • الخطوط المستخدمة: Cairo و IBM Plex Sans Arabic يدعمان RTL بشكل كامل

مرجع CSS

تعريف الخطوط في النظام:
/* النصوص العادية */
body {
  font-family: "IBM Plex Sans Arabic", system-ui, sans-serif;
  line-height: 1.85;
}

/* العناوين والأزرار */
h1, h2, h3, h4, h5, h6, .navbar-brand, .btn {
  font-family: "Cairo", system-ui, sans-serif;
  letter-spacing: 0;
}

/* أوزان العناوين */
h1, h2, h3 { font-weight: 800; }
h4, h5, h6 { font-weight: 700; }

/* النص المساعد */
.ap-muted {
  color: #64748b;
}

/* Responsive: الشاشات الصغيرة */
@media (max-width: 576px) {
  .navbar-brand {
    font-size: 1rem;
  }
}
